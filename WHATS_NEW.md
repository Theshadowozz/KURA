# What's New

This document explains the main differences between the project at the pre-elimination stage and the current version.

## Before

At the pre-elimination stage, the project was still very simple:
- It only had 3 main features:
  - Chatbot
  - Two-way communication
  - SEA map
- Language support was still limited to only 11 regional languages
- The UI structure and feature flow were still basic
- The documentation and overall presentation were not as complete as they are now

## Now

The current version is much more complete and mature:
- The total number of features has increased to 6
  - Streaming chatbot
  - Dictionary
  - Quick Speak
  - Two-way communication
  - SEA language map
  - Language quiz
- Language support has expanded to 24 regional languages
- The UI has been improved to feel cleaner, more responsive, and easier to use
- The backend and frontend systems are more integrated
- The project documentation is more complete, including the README, update log, architecture, and style guide
- The project is now closer to a functional MVP, not just an early prototype

## Main Difference

In short:
- Before: 3 features, 11 languages, early prototype
- Now: 6 features, 24 languages, more polished UI, better documentation, and much better readiness for a demo or handoff

## Most Noticeable Additions

- A Dictionary feature for language reference
- Quick Speak for fast text-to-speech
- The chatbot now streams responses instead of replying all at once
- Many more regional languages are supported
- The overall user experience looks more serious and presentable
