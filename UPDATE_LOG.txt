3. CULTURAL INFO CARD DI PETA
--------------------------------------------------------------------------------
Status  : NEW FEATURE
Files   : backend/main.py, frontend/src/App.jsx, frontend/src/styles.css

Backend (main.py):
  - Model MapAudioResponse ditambah 4 field baru:
      speakers     : jumlah penutur bahasa
      status       : status bahasa (Safe / Vulnerable / Endangered)
🟢 Safe (Aman)
Bahasa masih digunakan secara luas oleh semua generasi, diajarkan di sekolah, dan hadir di media. Contoh dalam dataset: Javanese (~82 juta penutur) dan Cebuano (~27 juta penutur).
🟡 Vulnerable / Safe but Declining
Bahasa masih digunakan, tapi generasi muda mulai beralih ke bahasa nasional atau bahasa yang lebih dominan. Jumlah penutur aktif berkurang dari waktu ke waktu. Contoh: Minangkabau, Iban, Lao.
🔴 Endangered (Terancam Punah)
Bahasa hanya digunakan oleh sebagian kecil komunitas, seringkali orang tua atau lansia. Anak-anak tidak lagi belajar bahasa ini sebagai bahasa pertama. Contoh: Mambae, Tutong, beberapa bahasa di Timor Leste dan Brunei.

      cultural_fact: satu fakta budaya dari field cultural_notes di JSON
      family       : rumpun bahasa (misal: Austronesian)
  - Endpoint /map-audio mengambil data tambahan dari sea_languages_knowledge.json
    setelah generate audio, menggunakan case-insensitive matching
  - Pengambilan cultural_fact: kalimat pertama dari field cultural_notes
  - Enrichment bersifat non-critical (error diabaikan agar audio tetap berfungsi)

Frontend (App.jsx):
  - handleMapClick menyimpan speakers, status, cultural_fact, family ke mapResult state
  - Map result card diperbarui dengan 4 lapis informasi:
      a. Header   : nama bahasa + negara + badge status berwarna
      b. Meta row : chips jumlah penutur (👥) dan rumpun bahasa (🌐)
      c. Greeting : label "GREETING" + teks salam dalam bahasa asli
      d. Fact     : kotak ungu berisi fakta budaya (💡)

CSS (styles.css):
  - .map-status-badge dengan 3 varian warna:
      .safe   : hijau (rgba 80,200,120)
      .warn   : kuning (rgba 255,180,50)
      .danger : merah (rgba 255,80,80)
  - .map-result-meta untuk chips row (speakers + family)
  - .map-meta-chip styling
  - .map-result-greeting + .map-greeting-label (uppercase label kecil)
  - .map-cultural-fact dengan background ungu transparan
  - .map-cultural-icon + paragraph styling
  - .map-result-top diperbarui: align-items flex-start, gap 12px
